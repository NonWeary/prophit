# Prophit 📈🔮
### *"The only prediction market for things that actually matter."*

Prophit is a satirical prediction market web app where users can bet fake money on absurd, unresolvable, and deeply important questions — like whether your roommate will do the dishes, or if the next Marvel movie will disappoint everyone.

Built with Next.js. Inspired by Polymarket. Funnier than both.

---

## Concept

Real prediction markets are serious financial instruments used to forecast geopolitical events, election outcomes, and economic indicators.

Prophit applies that same energy to whether Elon tweets something unhinged before 9am EST.

---

## Market Categories

**🏠 Daily Life**
- "Will my roommate do the dishes before Sunday?"
- "Will this meeting could have been an email?" *(resolves post-meeting)*
- "Probability the guy doing curls in the squat rack gets called out today"

**💻 Tech & Internet**
- "Will GPT-5 be announced before your therapist texts you back?"
- "Will this startup pivot before their Series A?"
- "Elon tweets something unhinged before 9am EST" *(perpetual daily market)*

**🎬 Pop Culture**
- "Which celebrity couple breaks up next?"
- "Will the reboot be described as 'a fresh take' or 'a beloved classic'?"
- "Chance the next Marvel movie scores below 50% on Rotten Tomatoes"

**🪞 Meta**
- "Will the user who just placed this bet regret it?"
- "Probability this site gets a cease-and-desist from a real prediction market"
- "Will Prophit raise a $20M seed round by EOY?"

---

## Features

- **Fake liquidity** — serious-looking charts for utterly stupid markets
- **AI Expert Analysis** — hedge-fund-memo-style commentary generated for every market
- **Unresolvable markets** — *"Will we ever know if hot dogs are sandwiches?"*
- **Badges** — *Diamond Hands, Wrong Again, Technically Correct*
- **Leaderboard of Worst Predictors** — celebrated, not shamed

## Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/prophit.git
cd prophit

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Fill in your Supabase + Auth keys

# Run locally
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see it.
